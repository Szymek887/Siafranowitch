Podstawowa struktura języka PHP

PHP to skryptowy język programowania służący do tworzenia stron i aplikacji internetowych. PHP jest językiem wykonywanym po stronie serwera, więc aby go użyć musimy użyć serwera (np. XAMPP) Korzystając z języka PHP nasz główny dokument strony musi mieć rozszerzenie .php (np. zamiast index.html -> index.pxp). Aby użyć skryptu PHP na stronie używamy tagu <?php --skrypt-- ?>

Utwórz podstawową strukturę do pisania skryptu w PHP, według poniższych wymagań:

  1. Zainstaluj i odpowiednio skonfiguruj serwer stron internetowych
  2. Utwórz dokument index.php z podstawową strukturą html
  3. Dodaj tag, który umożliwi pisanie skryptu php
  4. Dodaj instrukcję echo('Twoje imie'), która powinna wyświetlić

Czas na wykonanie zadania: 5 minut
