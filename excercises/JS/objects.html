Obiekty

Czym jest obiekt?

Obiekty to takie "pudełka". Na przykład samochód. Taki obiekt ma markę, model, pojemność silnika, rodzaj nadwozia, rodzaj paliwa, wymiary czy maksymalną prędkość. To są jego właściwości. Może jeździć, trąbić, świecą się jego światła. To jego czynności - w obiektowych językach programowania nazywamy je funkcjami lub metodami. Wszystkie te właściwości i czynności są upakowane w jeden obiekt - samochód. Dzięki temu jeżeli chcemy coś z nim zrobić, mamy dostęp do wszytkich rzeczy pod postacią jednego obiektu.


Analogicznie do powyższego przykładu obiektem jest każdy element na stronie. Na przykład przycisk, ma on swoje właściwości takie jak treść, czyli napis, który się na nim znajduje, wysokość, szerokość i pozycję na stronie.


Tworzenie obiektu

                
                    const car = {
                    brand: "Volkswagen",
                    model: "Golf",
                    generation: "VI",
                    maxSpeed: 240,
                    drive(){
                        return "The car is now driving"
                    }
                }

Odwoływanie się do właściwości obiektu

                
                console.log(car.brand) //Volkswagen

Powyżej zaprezentowane obiekty są proste. W pracy programisty jednak będziemy mieć do czynienia z obiektami złożonymi, których właściwościami będą kolejne obiekty. Na przykład człowiek, ma on swoje dane, ale dodatkowo na przykład ma psa, samochód, dom, telefon. Każdy z tych elementów ma swoje właściwości.

            
                const person = {
                name: "Adam",

                pet: {
                    name: "Mruczek",
                    type: "pies",
                    color: "black"
                    favoriteMeal: "mięso"
                    }

                car: {
                    brand: "Volkswagen",
                    model: "Golf",
                    generation: "VI"
                    }

                console.log(person.pet.name) //Mruczek

Dodawanie zmiennych jako właściwości obiektów

            
                let name = "Adam";
                let pet = "Mruczek";
                let carBrand = "Volkswagen";

                const person = {
                name: name,
                pet: pet,
                car: {
                    brand: carBrand
                    }
                }

                console.log(person.pet) //Mruczek

Wykorzystując wiedzę zdobytą na lekcji wykonaj poniższe polecenia:

W wyskakującym okienku poproś użytkownika o podanie:

  1. Imienia
  2. Nazwiska
  3. Numer telefonu
  4. Ulubionej marki samochodu
  5. Imienia zwierzaka
  6. Gatunku zwierzaka (w tej samej właściwości co poprzedni podpunkt)

A następnie wyświetl te dane na stronie w wyśrodkowanym divie w formie:

  1. Na samej górze: Imię i nazwisko w jednym paragrafie
  2. Poniżej numer telefonu: [podany przez użytkownika numer telefonu]
  3. Ulubiony samochód: [podana przez użytkownika marka]
  4. Ulubiony zwierzak: [podane przez użytkownika imię zwierzaka]
  5. Gatunek zwierzaka: [podany przez użytkownika gatunek zwierzaka]

Postaraj się, aby wyświetlony div wyglądał jak wizytówka

Czas na wykonanie zadania: 30 minut
