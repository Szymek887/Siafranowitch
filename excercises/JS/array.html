Tablice

Tablice służą do przechowywania wielu zmiennych. W języku JavaScript do pracy na tablicach można używać wbudowanego obiektu array. Posiada on metody do manipulacji tablicami zmiennych. Tablice w języku JavaScript zaczynają się od indeksu 0.


Tworzenie tablic 

                
                let Owoce = new Array(); // Tworzenie pustej tablicy za pomocą wbudowanej metody
                let Pieczywo = new Array(5); // Tworzeni pustej tablicy o wielkości 5

                let Warzywa = []; // Tworzenie pustej tablicy
                let Samochody = ["Fiat","Ford", "BMW"]; // Tworzenie tablicy z 3 elementami

Dodawanie wartości do tablicy

                
                let Warzywa = []; // Tworzenie pustej tablicy

                Warzywa.push("Bakłażan"); // Dodawanie elementu na ostatnie miejsce w tablicy
                console.log(Warzywa[0]); // Wypisze "Bakłażan"

                Warzywa[99] = "Marchewka"; // Dodawanie elementu na setne miejsce w tablicy 
                //(Tablice w JS zaczynają się od indeksu 0, więc 99 indeks to tak naprawdę setny element)

                Warzywa.unshift("Sałata"); // Dodanie elementu na początek tablicy
                console.log(Warzywa[0]) // Wypisze "Sałata"

Wykorzystując zdobytą wiedzę wykonaj poniższe zadanie:

  1. Stwórz tablicę o nazwie "Samochody"
  2. Dodaj do niej 3 marki samochodów w kolejności: Ford, Fiat, BMW
  3. Wypisz ostatni element tablicy
  4. Dodaj dowolny element na początek tablicy
  5. Wypisz z tablicy element "Fiat"
  6. Dodaj dowolny element na koniec tablicy

Czas na wykonanie zadania: 10 minut
