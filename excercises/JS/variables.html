Typy danych, zmienne i stałe

W języku JavaScript występuje 6 wbudowanych typów danych: null, undefined, Boolean, Number, String, Object. Zmienne służą do przechowywania danych i wyników w celu dalszego ich wykorzystania. W języku JavaScript zmienne deklaruje się za pomocą słowa kluczowego var lub let i const poprzedzającego nazwę zmiennej. Słowo var uznaje się za porzestarzałe.


Zmienne i stałe

                
                let a = 5; // Tworzenie zmiennej "a", o wartości 5

                const B = 3.14 // Tworzenie stałej, czyli zmiennej, której nie można zmienić wartości

null - reprezentuje zamierzony brak wartości elementu. 

                
                let a = null;
                console.log(a); // Wypisze "null"

undefined - każda zmienna stworzona za pomocą let lub var bez nadawanie wartości ma domyślnie wartość undefined 

                    
                    let a;
                    console.log(a); // Wypisze "undefined"

Boolean - przyjmuje wartość True lub False 

                    
                    let a = True;
                    let b = False;
                    console.log(a); // True
                    if(a){
                        console.log("Prawda") // Wypisze do konsoli
                    }

                    if(b){
                        console.log("Prawda") // Nie wypisze do konsoli
                    }

Number - zmienna o wartości liczbowej, np. 42 lub 3.14 

                
                let a = 5

                let b = 23
                let c = a+b

                console.log(c) // 28

String - tworzy obiek pozwalający działać na ciągu znaków. W JavaScrip-cie, ciągi znaków są niezmienialne.

                
                    let a = "Ala ma kota";
                    console.log(a.length) // Wypisuje długość ciągu

Object - jest typem złożonym. Wszystkie zmienne nie będące typem prymitywnym są obiektami (typami złożonymi). Charakteryzują się one tym, że zmienne nie mają przypisanej bezpośrednio wartości, a tylko wskazują na miejsce w pamięci, gdzie te dane są przetrzymywane. Obiektem jest na przykład Array.

                
                    const a = [1, 2, 3]; // Tworzenie obiektu

                    console.log(a) // [1, 2, 3]

Wykorzystując zdobytą wiedzę wykonaj poniższe zadanie:


  1. Stwórz zmienną typu String i przypisz do niej wartość "Hello World!"
  2. Stwórz stałą i nadaj jej wartość 3.14

Czas na wykonanie zadania: 5 minut
