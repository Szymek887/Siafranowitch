Switch

Czym instrukcja switch?

Switch jest rodzajem instrukcji warunkowej. Jest to instrukcja wyboru, a więc polega na porównywaniu wyniku do konkretnych wartości.

Sposób zastosowania instrukcji switch:


        const zmienna = prompt("Podaj zmienną");

switch (true) {
    case zmienna < 3:
        console.log("Liczba jest mniejsza od 3");
        break;
    case zmienna > 3 && zmienna < 7:
        console.log("Liczba jest większa od 3 i mniejsza od 7");
        break;
    case zmienna > 10:
        console.log("Liczba jest większa od 10");
        break;
    default:
        console.log("Żaden warunek nie jest spełniany");
}

Po każdym case należy umieścić klauzulę break, która zatrzyma wykonywanie instrukcji, w przeciwnym wypadku nawet po spełnieniu warunku program będzie próbował porównywać wyniki. Default to domyślny przypadek, jeśli żaden warunek nie będzie spełniony to będzie wykonana instrukcja znajdująca się w bloku default.


Wykorzystując zdobytą wiedzę wykonaj poniższe zadanie:

W oknie dialogowym poproś użytkownika o podanie nazwy zwierzaka domowego.

  1. Jeśli poda "kot" wyświetl napis "Koty są super!"
  2. Jeśli poda "pies" wyświetl napis "Psy są najlepsze!"
  3. Jeśli poda "chomik" wyświetl napis "Wszystkie chomiki są słodkie!"
  4. W domyślnym przypadku wyświetl napis "Jakiś dziwny ten zwierzak"
