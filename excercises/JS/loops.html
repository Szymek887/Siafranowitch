Pętla for, while, do while

Czym jest pętla?

Pętla to taka konstrukcja, która służy do zdefiniowania instrukcji, które będą powtarzane wielokrotnie, do spełnienia jakiegoś warunku. Jeśli na przykład chcemy wypisać liczby od 0 do 100, to nie piszemy stu linijek kodu, tylko stosujemy pętlę, która w kilku linijkach wykona to samo.

Jeżeli wiemy, ile razy ma się wykonać określona część kodu, korzystamy z pętli for:


        for (let i = 0; i < 100; i++) {
            //kod, który będzie wykonywany dopóki warunek jest spełniany
        }

Jeżeli nasz kod ma się wykonywać dopóki jakiś warunek będzie spełniany, korzystamy z pętli while:


        while (i < 100) {
            //kod, który będzie wykonywany dopóki warunek jest spełniany
            //tutaj należy pamiętać, aby zmieniać zmienną i, na przykład inkrementując
        }

Jeżeli chcemy, aby nasz kod wykonał się conajmniej raz, korzystamy z konstrukcji do while:


        do {
            //kod, który będzie wykonywany dopóki warunek jest spełniany
            //tutaj należy pamiętać, aby zmieniać zmienną i, na przykład inkrementując
        } while(i<100);
