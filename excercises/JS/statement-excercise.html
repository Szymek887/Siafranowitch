Instrukcje warunkowe

Czym są instrukcje warunkowe?

Instrukcja warunkowa jest rozgałęzieniem programu. W zależności od tego, czy warunek zawarty w instrukcji jest prawdziwy lub fałszywy, wykonane zostaną różne instrukcje. W naszym programie możemy użyć klauzuli else, w której umieścimy fragment kodu, który wykona się, gdy warunek w if nie zostanie spełniony.

Poniżej znajduje się sposób wykorzystania instrukcji warunkowej:


        if(warunek_logiczny) {
            //kod jeśli warunek zostanie spełniony
        }

        else {
            //kod jeśli warunek nie zostanie spełniony
        }

Warunek logiczny


        //W klauzuli if umieszczamy warunek logiczny, czyli:
        if(a == b){}
        if(a != b){}
        if(a > b){}
        if(a < b){}

        //Warunki możemy łączyć za pomocą operatorów || (OR) oraz && (AND), na przykład:
        if(a == b && a != c){}
        if(a > b || a > c){}

Wykorzystując zdobytą wiedzę wykonaj poniższe zadanie:

W wyskakującym okienku poproś użytkownika o podanie liczby od 0 do 10. Po podaniu liczby::

  1. Jeśli liczba jest mniejsza od 5 wypisz: "Liczba jest mniejsza od 5"
  2. Jeśli liczba jest większa od 5 wypisz: "Liczba jest większa od 5:"
  3. Jeśli liczba jest mniejsza od 4 i mniejsza lub równa 6 wypisz: "4 < {liczba} <= 6"
