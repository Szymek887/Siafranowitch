Selektory CSS - rozwiązanie

Wykorzystując zdobytą na lekcji wiedzę wykonaj poniższe zadanie:

Utwórz podstawową strukturę strony WWW, uwzględniając poniższe wymagania:

  1. Dodaj nagłówek stopnia drugiego i ostyluj go dowolnym sposobem tak aby miał kolor biały, pogrubioną czcionkę oraz podkreślenie
  2. Dodaj dowolne zdjecie z internetu i dodaj mu id "image", ustaw w CSS wysokość i szerokość na 300px
  3. Dodaj dwa paragrafy, jeden o id "first-p", drugi "second-p" i ostyluj je tak samo wykorzystując jak najmniej linijek kodu css. Ustaw wielkość czcionki na 18px, wysokość lini na 20px

Czas na wykonanie zadania: 20 minut
